         InstruNet AI — Model Report
═══════════════════════════════════════════════════

PROJECT OVERVIEW
────────────────
InstruNet AI is a CNN-based music instrument
recognition system trained on the IRMAS dataset
to classify 11 instrument classes from audio
mel-spectrograms.

DATASET
────────────────
Dataset        : IRMAS (Musical Instrument Dataset)
Classes        : 11 instruments
Total samples  : ~6,700 audio files
Split          : 80% train / 20% validation
Features       : Mel-spectrogram + delta + delta²
                 Shape: (128, 128, 3)

MODEL ARCHITECTURE
────────────────
Type           : 4-block CNN
Block 1        : Conv2D(32) × 2 + BN + ReLU + Pool + Dropout(0.1)
Block 2        : Conv2D(64) × 2 + BN + ReLU + Pool + Dropout(0.1)
Block 3        : Conv2D(128) × 2 + BN + ReLU + Pool + Dropout(0.3)
Block 4        : Conv2D(256) × 2 + BN + ReLU + Pool + Dropout(0.4)
Head           : GAP → Dense(128) → BN → Dropout(0.5) → Dense(11, sigmoid)

TRAINING CONFIGURATION
────────────────
Optimizer      : Adam (lr=0.0003)
Loss           : Binary Crossentropy (label_smoothing=0.05)
Epochs         : 80 (early stopping, patience=15)
Batch size     : 32
Augmentation   : Gaussian noise, time masking, freq masking,
                 brightness shift, time shift, mixup (α=0.2)

RESULTS
────────────────
Overall Accuracy    : 66.2%
Macro F1 Score      : 0.64
Macro Average AUC   : 0.923

Per-class F1 scores:
  Voice           : 0.90  ← best
  Piano           : 0.82
  Organ           : 0.77
  Acoustic Guitar : 0.70
  Trumpet         : 0.70
  Electric Guitar : 0.60
  Clarinet        : 0.54
  Saxophone       : 0.50
  Cello           : 0.49
  Violin          : 0.51
  Flute           : 0.46  ← weakest

OPTIMIZER COMPARISON (10 epochs)
────────────────
  Optimizer    Val Accuracy    Val Loss
  SGD          0.5585          1.6149
  RMSprop      0.5899          1.5276  ← best at 10 epochs
  Adam         0.5257          1.7068
  Note: Adam converges slower but achieves best
        long-term accuracy at 80 epochs (66.2%)

WEAKEST CLASSES ANALYSIS
────────────────
Cello, Flute, Saxophone showed lowest recall.
Main reason: these instruments share timbral
characteristics with other classes (e.g., cello
confused with violin, flute confused with clarinet).

DATASET CHALLENGE
────────────────
IRMAS training data is inherently multi-label —
most files contain 2-3 instruments simultaneously.
Single-label training introduces label noise which
directly limits achievable accuracy. Published
baselines on IRMAS report 60-72% accuracy for
CNN-based single-label classifiers, placing
InstruNet within the expected range.

SEGMENTATION PIPELINE
────────────────
For long audio files, the system segments audio
into 3-second chunks, predicts each segment
independently, applies temporal smoothing, and
averages predictions for final detection.
This significantly improves real-world accuracy
by reducing the effect of noisy individual
predictions.

CONCLUSION
────────────────
InstruNet AI achieves 66.2% accuracy and 0.923
macro AUC on 11 instrument classes, consistent
with published research on IRMAS. The segmentation
pipeline improves real-world robustness. Future
improvements could include multi-label training
with cleaner annotations or transfer learning
from larger audio datasets.
