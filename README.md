# 🎵 InstruNet AI — CNN-Based Music Instrument Recognition

Automatically detects musical instruments in audio tracks using CNNs + Mel-Spectrograms.

---

## 📁 Project Structure

```
instrunet/
├── phase1_preprocessing.py   # Audio → Mel-Spectrogram images
├── phase2_cnn_model.py        # Build & Train CNN
├── phase3_evaluation.py       # Evaluate, confusion matrix, tuning
├── phase4_app.py              # Streamlit dashboard
├── requirements.txt
├── data/
│   ├── IRMAS-TrainingData/    ← Put your IRMAS dataset here
│   └── spectrograms/          ← Auto-generated by Phase 1
└── models/
    ├── instrunet_cnn.h5       ← Saved after Phase 2
    ├── label_classes.json     ← Instrument labels
    └── evaluation_results.json
```

---

## 🚀 How to Run (Step by Step)

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Set up IRMAS dataset
- Download from: https://www.upf.edu/web/mtg/irmas
- Extract and place in: `data/IRMAS-TrainingData/`
- Folder structure inside should be:
  ```
  IRMAS-TrainingData/
  ├── cel/   (Cello)
  ├── cla/   (Clarinet)
  ├── flu/   (Flute)
  ├── gac/   (Acoustic Guitar)
  ├── gel/   (Electric Guitar)
  ├── org/   (Organ)
  ├── pia/   (Piano)
  ├── sax/   (Saxophone)
  ├── tru/   (Trumpet)
  ├── vio/   (Violin)
  └── voi/   (Voice)
  ```

### 3. Phase 1 — Preprocessing
```bash
python phase1_preprocessing.py
```
Generates mel-spectrogram PNG images in `data/spectrograms/`

### 4. Phase 2 — Train CNN
```bash
python phase2_cnn_model.py
```
Trains for up to 30 epochs, saves best model to `models/instrunet_cnn.h5`
Expected accuracy: ~75–85%

### 5. Phase 3 — Evaluate & Tune
```bash
python phase3_evaluation.py
```
Generates confusion matrix, per-class accuracy chart, evaluation JSON

### 6. Phase 4 — Launch Dashboard
```bash
streamlit run phase4_app.py
```
Opens web dashboard at http://localhost:8501
