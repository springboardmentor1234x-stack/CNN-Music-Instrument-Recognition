IRMAS Dataset Description

This project uses the IRMAS dataset (Instrument Recognition in Musical Audio Signals).

The dataset contains recordings of 11 musical instruments:

cel – cello  
cla – clarinet  
flu – flute  
gac – acoustic guitar  
gel – electric guitar  
org – organ  
pia – piano  
sax – saxophone  
tru – trumpet  
vio – violin  
voi – voice  

Each folder contains WAV audio recordings of the instrument.

These audio files are converted into Mel spectrogram images for CNN training.
